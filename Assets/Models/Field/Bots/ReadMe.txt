Стоячие Idle анимации бледятся с любыми из четырёх Action анимациями (положения ног совпадает, остальное не очень)

Разговорные анимации совпадают полностью

Только для двух из трёх сидячих Idle анимаций есть Action анимации. Комбинируются между собой согласно названиям (Idle_sit_1 стыкуется только с Action_for_Idle_sit_1)

Анимация Squatting_position_1 используется для бота на лесах, который сидит рядом с игроком, анимация Squatting_position_2 для бота, который сидит на противоположной стороне
